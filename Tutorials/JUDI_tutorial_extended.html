1. Introduction to Seismic Imaging

Seismic imaging techniques like Full-Waveform Inversion (FWI) and Reverse-Time Migration (RTM) are essential tools for understanding subsurface geologyâ€”critical for projects like Geological Carbon Storage (GCS).

Key Concepts:
- FWI: Iteratively update a velocity model by minimizing the difference between observed and modeled seismic data.
- RTM: Use the time-reversed seismic wavefield to reconstruct an image of the subsurface reflectors.

JUDI.jl leverages Juliaâ€™s efficient computation capabilities to perform seismic modeling, imaging, and inversion with ease.

Tip

Why JUDI.jl?
- Efficient computations for seismic imaging.
- Easy-to-use interface for FWI, RTM, and time-lapse monitoring.
- Scalable for large geological datasets.

3. Seismic Forward Modeling

Theory

Forward modeling simulates seismic wave propagation governed by the acoustic wave equation:

\[ \frac{\partial^2 u}{\partial t^2} = v^2 \nabla^2 u + s(t) \]

where:
- ( u(t,x,z) ) = pressure wavefield
- ( v ) = velocity model (m/s)
- ( s(t) ) = source term
- ( ^2 ) = Laplacian operator

This equation describes how seismic waves propagate through the Earth based on known velocity structures.

3.1 Defining a Simple Velocity Model

using JUDI, SeismicJulia, PyPlot

# Grid setup: 2D model with uniform spacing
nx, nz = 100, 100  # Grid points in x and z
dx, dz = 10.0, 10.0  # Spacing in meters
model = Model((nx, nz), (dx, dz), vp=1500.0)

# Add a velocity anomaly
for i in 40:60, j in 40:60
    model.vp[i, j] = 2000.0
end

# Display the model
imshow(model.vp', cmap="viridis")
colorbar(); title("Velocity Model (m/s)")

Explanation:
- A homogeneous model with a higher-velocity anomaly in the center simulates a carbonate reef or similar geological structure.

Expected Plot:
- Velocity model with a visible high-velocity anomaly.

3.2 Generating Synthetic Seismic Data

# Ricker wavelet with peak frequency of 25 Hz
wavelet = ricker_wavelet(25.0, dt=0.001, duration=2.0)

# Define source and receiver locations
src = Source((50, 5), wavelet)
rec = ReceiverLine((5, 95), nx=20, spacing=5)

# Forward modeling
data = judi_forward(model, src, rec)

# Visualize the synthetic data
imshow(data', cmap="seismic", aspect="auto")
xlabel("Receiver index"); ylabel("Time index")
title("Synthetic Seismic Data")

Key Parameters:
- Ricker Wavelet: Common choice for seismic modeling.
- Source & Receiver: The source at the surface and a receiver line near the surface.

Expected Plot:
- Seismic data as a time vs receiver index image.

Citation:
- Aki & Richards (2002). Quantitative Seismology.

4. Reverse-Time Migration (RTM)

Theory

RTM uses the two-way wave equation and time-reversal principles to image subsurface structures:

\[ \nabla^2 u = \frac{1}{v^2}\frac{\partial^2 u}{\partial t^2} \]

The imaging condition is applied by cross-correlating the forward and backward wavefields:

\[ I(x,z) = \int u_{forward}(t,x,z) u_{backward}(t,x,z) dt \]

Code Implementation

# Run RTM
image = judi_rtm(model, data, src, rec)

# Plot the subsurface image
imshow(image', cmap="gray", aspect="auto")
title("RTM Image")

Explanation:
- Forward propagate the source wavefield.
- Backward propagate the recorded data.
- Apply imaging condition.

Expected Plot:
- An image of the reflectors beneath the surface.

Citation:
- Baysal et al. (1983). Reverse-time Migration.

5. Full-Waveform Inversion (FWI)

Theory

FWI solves an optimization problem to minimize the misfit between observed and predicted seismic data:

\[ \underset{m}{\text{minimize}} \; \Phi(m) = \frac{1}{2}\|d_{obs} - d_{pred}(m)\|^2 \]

where:
- ( m ) = model parameters (e.g., velocity)
- ( d_{obs} ) = observed seismic data
- ( d_{pred}(m) ) = predicted data from the model

The gradient is computed via the adjoint-state method.

Code Implementation

using Optim

# Initial guess: homogeneous model
initial_model = Model((nx, nz), (dx, dz), vp=1500.0)

# Loss function (misfit)
function misfit(m)
    model = Model((nx, nz), (dx, dz), vp=reshape(m, nx, nz))
    pred = judi_forward(model, src, rec)
    return sum((pred - data).^2)
end

# Optimize using LBFGS
result = optimize(misfit, vec(initial_model.vp), LBFGS())

# Update the model
fwi_model = reshape(result.minimizer, nx, nz)

# Plot the updated model
imshow(fwi_model', cmap="viridis")
title("FWI-Inverted Velocity Model")

Explanation:
- Compute the misfit and use LBFGS to find the optimal velocity model.
- Gradients are computed via the adjoint-state method.

Expected Plot:
- Improved velocity model with the anomaly correctly positioned.

Citation:
- Virieux & Operto (2009). An overview of full-waveform inversion in exploration geophysics.

6. Time-Lapse Seismic Monitoring for Carbon Storage

Concept

Monitoring CO2 storage sites involves comparing seismic surveys over time to track plume migration:

\[ d_{diff} = d_{monitor} - d_{baseline} \]

where:
- ( d_{baseline} ) = initial survey data
- ( d_{monitor} ) = subsequent survey data

Code Implementation

# Simulate baseline and monitor surveys
baseline = judi_forward(model, src, rec)
model.vp[40:60, 40:60] .= 1800.0  # Simulating CO2 plume
monitor = judi_forward(model, src, rec)

# Calculate time-lapse difference
difference = monitor - baseline

# Display the time-lapse differences
imshow(difference', cmap="seismic", aspect="auto")
title("Time-Lapse Seismic Data (CO2 Plume Detection)")

Insight:
- The time-lapse difference identifies areas of CO2 migration.

Expected Plot:
- Time-lapse differences highlighting the CO2 plume.

Citation:
- Chadwick et al. (2010). Monitoring CO2 storage sites.
